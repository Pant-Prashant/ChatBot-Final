# 🤖 AI Chatbot

A full-stack AI chatbot application built with **React**, **FastAPI**, **PostgreSQL**, and **LangChain**. The application provides a modern chat interface with secure user authentication, persistent conversation history, and AI-powered responses.

---

## 🚀 Features

- 🔐 User Authentication (Sign Up & Login)
- 💬 AI-powered chatbot using LangChain
- 📁 Multiple conversations support
- 📝 Persistent chat history
- ⚡ FastAPI backend
- 🎨 Responsive React frontend
- 🗄️ PostgreSQL database
- 🔒 Password hashing for secure authentication
